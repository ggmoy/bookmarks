## How to add a bookmark

1. Find the matching category (`##`) in the index, or create a new one and add it to the index.
2. Within the category, use a subheading (`###`) per site or topic to group related links.
3. Add the link using the format `- [Title](URL)`. Use indented sublists for related links.
